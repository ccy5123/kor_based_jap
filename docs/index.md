# Korean → Japanese IME

An Android Input Method Editor that lets you type Japanese kana by
typing Korean Hangul jamo on a 두벌식 (Beolsik) or 천지인 (Cheonjiin)
keyboard.  Powered by an offline Mozc-derived dictionary + Viterbi
engine — no network calls, all processing on-device.
