# KorJpnIme v1.0.0

初回公開リリース — 韓国語 2-beolsik キーボードレイアウトで日本語を入力
できる Windows TSF IME です。Mozc レベルの漢字変換品質を提供します。

> 🌏 Languages: [English](RELEASE_NOTES_v1.0.0.md) · [한국어](RELEASE_NOTES_v1.0.0.ko.md) · **日本語**

## インストール

1. `KorJpnIme-v1.0.0.zip` をダウンロード (下記に添付)
2. 任意のフォルダに解凍
3. 管理者権限の PowerShell で:
   ```powershell
   powershell -ExecutionPolicy Bypass -File install.ps1
   ```
4. **ログアウトして再ログイン** (ctfmon が新しい TIP をクリーンにロードするため必須)
5. 設定 → 時刻と言語 → 言語 → 韓国語 → 言語のオプション → キーボードを追加
   → *Korean-Japanese IME* を選択
6. 日本語を入力したい時は `Win+Space` で切り替え

詳細は同梱の `README.md` と `CHANGELOG.md` を参照してください。

## 主な機能

- **韓国語 2-beolsik → 日本語かな** — 複合母音 (ㅗ+ㅏ=ㅘ → わ)、複合
  パッチム (ㄴ+ㅈ=ㄵ)、撥音 (ん) / 促音 (っ) の自動認識
- **Mozc OSS 辞書** — 約 75 万かなキー / 129 万エントリ、品詞 ID (lid/rid)
  とコスト値を含む
- **Top-K Viterbi 分節** — Mozc の 2672×2672 bigram 接続コスト行列を使用。
  「わたしの」のような複合形が `私 + の` に正しく分節され、**「私の」が
  単一候補として** 上位に提示されます
- **動詞活用のサポート** — Mozc の suffix テーブル (る / た / ます / ない
  等) により たべます / よみました / きれいです などが正しく分節されます
- **外来語の自動カタカナ変換** — 漢字経路が無い場合でも `한바아가아` に
  対して候補リストに `ハンバーガー` が自動で提供されます
- **3 つの特殊キーパターン** — 韓国語で意味を持たない入力シーケンスを
  日本語助詞に再利用:
  - `ㅇ-ㅗ-ㅗ` → を (目的格)
  - `ㅇ-ㅘ-ㅏ` → は (主題)
  - `ㅇ-ㅔ-ㅔ` → へ (方向)

  日本語の長母音 (おお、ええ 等) には影響なし — こちらは `ㅇ-ㅗ-ㅇ-ㅗ`
  のように母音間に明示的な ㅇ が必要
- **ユーザー学習辞書** — 選択した漢字が次回から上位に浮上。LFU 方式で
  古いエントリを自動プルーニング (デフォルト 5000 エントリ上限、
  `settings.ini` で調整可能)
- **設定ファイルのホットリロード** — `%APPDATA%\KorJpnIme\settings.ini`
  を編集すると次のキーストローク時に即座に反映 (ログアウト不要)
- **カタカナモードトグル** — デフォルト `RAlt+K` (設定ファイルで変更可能)
- **MS-IME / Mozc スタイルの preedit** — 点線の青下線で変換中テキストを表示

## 既知の制限

- **再変換 (`ITfFnReconversion`) 未実装** — 既に確定された日本語を選択
  して再変換する機能は未対応。回避策: 削除して再入力
- **初回インストール / アップデート時にログアウト → ログインが必要** —
  ctfmon が TIP DLL をセッション単位でキャッシュする TSF の仕様
- **ユーザー学習は最初の segment の順位付けのみに影響** — viterbi path
  cost 自体には学習結果が反映されない (今後の改善予定)
- **Top-K は K=5 にハードコード** — settings.ini への公開予定
- **Windows 11 で主に検証** — Windows 10 も動作予定だが CI では未実行

## バンドルされた外部データ

`jpn_dict.txt` / `kj_dict.bin` / `kj_conn.bin` は Google Mozc OSS
(BSD-3) 由来で、Mozc データ自体も NAIST IPAdic および沖縄パブリック
ドメイン辞書から派生しています。完全な帰属表示は同梱の
`dict/LICENSES.txt` に含まれており、データファイルと共に再配布する際は
必ず同梱してください。
